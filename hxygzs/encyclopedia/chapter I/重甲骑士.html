重甲骑士
生命值 : 80 (护甲无敌)
攻击伤害 : 普通伤害
特殊效果 : 无法通过劈砍BOSS护甲恢复TP
被动 :
血量低于40时技能会附带陨石(普通伤害)
在硬直时护甲无效
硬直时不会自动解除硬直,造成20伤害(伤害将无法大于20)后解除硬直
主动 :
向高处跳起,狠狠砸向地面,造成地震波(0.5高),造成普通伤害
蓄力(2s)向前造成地震波(0.5高),造成普通伤害
向前冲刺一段距离,造成普通伤害
举起武器,向周围范围攻击,造成普通伤害
硬直判定 :
攻击次数 : 15
连击次数 : -1
硬直上限 : 999
背景故事 :
重甲骑士本是弱小的人,因得到了神秘的盔甲而变得非常强大,是泽雷嘉最得意的徒弟。
先人笔记 :
这套盔甲几乎无法被破坏,我也想来一套,只可惜太笨重了。
幻影低语 :
"我是强大的骑士,我的盔甲坚不可摧！"
"我要变强！"
"没有人能阻止我！"
